# GameHub

App for listing games with platform, genres, order, search, using Vite + React + TypeScript.

## Usage

Install dependencies

```
npm i
```

Change API key [RAWG API](https://rawg.io/) in file (/src/services/api-client.ts)

Execute mode development

```
npm run dev
```

Build the app

```
npm run build
```
